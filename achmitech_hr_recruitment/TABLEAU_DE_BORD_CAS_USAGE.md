# Cas d'Usage du Tableau de Bord de Recrutement

## 1. Suivi des Candidatures en Temps Réel

**Objectif** : Avoir une vision d'ensemble de l'état de toutes les candidatures

**Étapes** :
1. Allez à **Recrutement** → **Tableau de Bord**
2. Consultez la section "Candidatures par Étape"
3. Cliquez sur **Vue Kanban** pour une vue par étape
4. Déplacez les candidatures entre les colonnes selon leur progression

**Résultat** : Vous avez une vue claire du pipeline de recrutement

---

## 2. Analyser les Goulots d'Étranglement

**Objectif** : Identifier où les candidatures sont bloquées

**Étapes** :
1. Consultez le graphique "Candidatures par Étape"
2. Identifiez l'étape avec le plus grand nombre de candidatures
3. Cliquez sur cette étape pour voir les candidatures
4. Investiguer les raisons du blocage

**Résultat** : Vous pouvez prendre des mesures pour débloquer le processus

---

## 3. Suivre les Performances par Poste

**Objectif** : Comparer l'efficacité du recrutement pour différents postes

**Étapes** :
1. Consultez le graphique "Top Postes"
2. Identifiez les postes avec le plus de candidatures
3. Cliquez sur le graphique pour voir le détail par poste
4. Utilisez la vue Pivot pour une analyse croisée (Poste × Étape)

**Résultat** : Vous comprenez quels postes attirent le plus de candidatures

---

## 4. Évaluer la Qualité des Candidatures

**Objectif** : Analyser la distribution des scores d'évaluation

**Étapes** :
1. Consultez "Distribution des Évaluations"
2. Observez la répartition des scores :
   - Beaucoup de "Très favorable" = bon processus de sélection
   - Beaucoup de "À évaluer" = besoin d'actions de suivi
3. Identifiez les évaluations incomplètes

**Résultat** : Vous pouvez ajuster vos critères d'évaluation

---

## 5. Gérer les Entretiens

**Objectif** : Planifier et suivre les entretiens de la semaine

**Étapes** :
1. Cliquez sur "Entretiens Cette Semaine"
2. Consultez la liste des entretiens programmés
3. Vérifiez que tous les interviewers sont disponibles
4. Envoyez des rappels si nécessaire

**Résultat** : Aucun entretien n'est oublié

---

## 6. Identifier les Candidatures Non Évaluées

**Objectif** : Assurer un suivi des candidatures sans retard

**Étapes** :
1. Cliquez sur "Candidatures Non Évaluées"
2. Consultez la liste des candidatures sans évaluation
3. Planifiez les entretiens ou demandez des évaluations
4. Mettez à jour les dossiers

**Résultat** : Chaque candidature a un suivi approprié

---

## 7. Rapport Mensuel de Recrutement

**Objectif** : Générer un rapport sur la performance du mois

**Étapes** :
1. Consultez le **Taux de Conversion**
2. Notez le **Total des Candidatures** du mois
3. Analysez la distribution par étape
4. Identifiez les tendances

**Données clés à rapporter** :
- Total de candidatures reçues
- Nombre d'embauches
- Taux de conversion
- Durée moyenne du processus
- Postes les plus demandés

---

## 8. Optimisation du Processus de Recrutement

**Objectif** : Améliorer continuellement votre processus

**Questions à poser** :
- Où perdez-vous le plus de candidats ?
- Quel est votre taux d'abandon par étape ?
- Quel est le temps moyen pour chaque étape ?
- Quels postes sont les plus difficiles à pourvoir ?

**Actions possibles** :
1. Simplifier les étapes trop longues
2. Accélérer les retours aux candidats
3. Améliorer la communication
4. Adapter les critères d'évaluation

---

## Métriques Clés à Suivre

| Métrique | Formule | Objectif |
|----------|---------|----------|
| Taux de Conversion | (Embauches / Candidatures) × 100 | > 5% |
| Durée du Processus | Date embauche - Date candidature | < 30 jours |
| Taux d'Abandon par Étape | (Candidats éliminés / Candidats reçus) × 100 | < 50% par étape |
| Satisfaction du Manager | Feedback qualitatif | > 8/10 |
| Coût par Embauche | Budget recrutement / Nombre embauches | À définir |

## Dépannage

### Problème : Les données ne se mettent pas à jour
**Solution** : Actualisez la page (F5) ou videz le cache du navigateur

### Problème : Je ne vois pas certaines candidatures
**Solution** : Vérifiez les droits d'accès, consultez le gestionnaire de permissions

### Problème : Les évaluations ne s'affichent pas
**Solution** : Assurez-vous que les évaluations ont une date renseignée
